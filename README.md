## ⚡ Commands

| Command | Description |
|---------|-------------|
| <h1 align="center">💀 `$nuke`</h1> | <b>Xóa toàn bộ channel → Tạo channel mới → Tạo webhook → Flood mention → DM spam toàn bộ member → Quảng bá kênh</b> |
| <h1 align="center">🔨 `$ban`</h1> | <b>Mass ban toàn bộ members trong server với tốc độ cực nhanh</b> |

---

## 📦 Cài đặt & Chạy

```bash
# 🚀 Clone về máy
git clone https://github.com/tuanhaidepzai/discord-n3ke-bot
cd discord-n3ke-bot

# 📦 Cài dependencies
pip install -r requirements.txt

# 🎮 Chạy bot
python main.py
```
